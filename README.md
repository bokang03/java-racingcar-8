# java-racingcar-precourse

## 구현 목록

### 입출력 예시
##### 입력
- 경주할 자동차 이름(이름은 쉼표(,) 기준으로 구분)
```
pobi,woni,jun
```
- 시도할 횟수
```
5
```

##### 출력
- 차수별 실행 결과
```
pobi : --
woni : ----
jun : ---
```
- 단독 우승자 안내 문구
```
최종 우승자 : pobi
```
- 공동 우승자 안내 문구
```
최종 우승자 : pobi, jun
```
- 실행 결과 예시
```
경주할 자동차 이름을 입력하세요.(이름은 쉼표(,) 기준으로 구분)
pobi,woni,jun
시도할 횟수는 몇 회인가요?
5

실행 결과
pobi : -
woni : 
jun : -

pobi : --
woni : -
jun : --

pobi : ---
woni : --
jun : ---

pobi : ----
woni : ---
jun : ----

pobi : -----
woni : ----
jun : -----

최종 우승자 : pobi, jun
```

### 기능
- [x] 자동차 이름을 쉼표(,)를 기준으로 구분하는 기능
- [x] 각 자동차에 0~9 사이에서 무작위 값 생성하는 기능
- [x] 무작위 값이 4 이상이면 전진 4미만이면 정지하는 기능 
- [x] 각 횟수마다 자동차의 전진 수 기록하는 기능
- [x] 우승자 판별 기능

### 입력
- [x] 경주할 자동차 이름 입력
- [x] 경주 횟수 입력

### 출력 
- [x] 경주할 자동차 이름 입력 문구 출력
- [x] 경주 횟수 입력 문구 출력
- [x] 경주를 진행할 때마다 해당 자동차 이름 및 위치 출력
- [x] 최종 우승자 출력

### 예외 처리
- [x] 자동차 이름 입력 시
  - [x] 자동차 이름이 5자를 초과할 경우
  - [x] 자동차 이름이 중복될 경우
  - [x] 쉼표를 연속해서 사용할 경우 
  - [x] 이름을 공백으로 입력할 경우 
- [x] 경주 횟수 입력 시 
  - [x] 양의 정수가 아닌 경우
